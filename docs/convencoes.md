## Símbolos NTSK (prefixos de wikilinks)

| Símbolo | Significado | Subgrafo gerado |
|---------|-------------|-----------------|
| `*`     | Ref. especialmente clara | — |
| `✓`     | Ref. criticamente pertinente | — |
| `+`     | Coleção mais completa | — |
| `✡`     | Cumprimento profético | Grafo tipológico AT→NT |
| `▶`     | Citação AT→NT | Grafo de citações |
| `◐`     | Contraste doutrinário | Relação CONTRASTA |
| `=`/`⩲` | Tipo / Antítipo | Grafo tipológico |
| `‡`     | Doutrina falsa | Aviso em conceitos |
