## Símbolos e Abreviaturas no The New Treasury of Scripture Knowledge (NTSK)

Esta tabela fornece explicações dos símbolos e abreviaturas usados nesta edição do The New Treasury of Scripture Knowledge, projetada para uso no Obsidian.

| **Símbolo/Abreviatura** | **Significado**                                | **Notas/Uso**                                                                                                        |
| ----------------------- | ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **\***                  | Referência Especialmente Clara                 | Priorize estas referências, especialmente para novos usuários.                                                       |
| **✓**                   | Referência Criticamente Clara/Pertinente       | Referências altamente significativas; essenciais para o entendimento.                                                |
| **+**                   | Coleção Mais Completa de Referências           | Indica um conjunto mais extenso de referências ao tópico no versículo.                                               |
| **+* ou +✓**            | Referências Adicionais/Mais Completas          | Indica ainda mais referências ou uma coleção mais completa para o tópico.                                            |
| **◐**                   | Contraste                                      | Identifica diferentes pontos de vista, contradições ou posições doutrinárias alternativas.                           |
| **=**                   | Tipo ou Antítipo                               | Indica uma relação simbólica entre figuras/eventos do Antigo e Novo Testamentos.                                     |
| **⩲**                   | Tipo/Antítipo (Autoridade Bíblica)             | Tipo ou antítipo identificado com base nas Escrituras.                                                               |
| **▶**                   | Citação (AT para NT)                           | Indica uma citação no Novo Testamento proveniente do Antigo Testamento. Também mostra passagens do AT citadas no NT. |
| **▶𝒫**                 | Citação (Pentateuco nos Profetas)              | Identifica citações do Pentateuco encontradas nos livros proféticos.                                                 |
| **✡**                   | Cumprimento da Profecia                        | Indica uma referência que cumpre uma profecia.                                                                       |
| **∥**                   | Passagem Paralela Estrita                      | Encontrada nos evangelhos, em Samuel, Reis e Crônicas. Não é exaustiva, mas seletivamente anotada.                   |
| **❅S#**                 | Número de Strong (Hebraico)                    | Todas as ocorrências da palavra original hebraica estão listadas. (ex: ❅S#2313h)                                     |
| **✣S#**                 | Número de Strong (Grego)                       | Todas as ocorrências relevantes/similares da palavra grega estão listadas. (ex: ✣S#2313g)                            |
| **( )**                 | Figura de Linguagem – Não Aparentes em Inglês  | Indica uma figura de linguagem não facilmente visível nas traduções KJV, YLT ou Rotherham.                           |
| **( )**                 | Tradução da Palavra Hebraica/Grega             | Tradução da palavra hebraica/grega fornecida quando ela difere em uma referência específica.                         |
| **( )**                 | Palavra Suprida (Elipse)                       | Palavra não presente na língua original, mas suprida de acordo com a figura de linguagem.                            |
| **CB**                  | A Bíblia de Companhia                          | Referência à A Bíblia de Companhia.                                                                                  |
| **F/L**                 | Primeiro/Último (Isaías)                       | Referências às seções de Isaías (1-39 e 40-66) demonstrando a unidade do livro.                                      |
| **ƒ**                   | Identificador da Figura de Linguagem           | Figura de linguagem numerada, referenciando sua explicação e outras instâncias (ex: ƒ102).                           |
| **B**                   | Bullinger – Figuras de Linguagem               | Referência ao número da página em E.W. Bullinger, *Figuras de Linguagem Usadas na Bíblia* (ex: B542).                |
| **g ou h**              | Referência Verbal (Hebraico/Grego)             | Indica referência à mesma palavra hebraica/grega após uma referência cruzada.                                        |
| **Gr.**                 | Grego                                          |                                                                                                                      |
| **Heb.**                | Hebraico                                       |                                                                                                                      |
| **ISBE**                | Enciclopédia Bíblica Padrão Internacional      |                                                                                                                      |
| **lit.**                | Literalmente                                   |                                                                                                                      |
| **mg**                  | Leitura Marginal                               | Referência às leituras marginais na versão KJV/Autorizada.                                                           |
| **MM**                  | Moulton & Milligan                             | James Hope Moulton e George Milligan, *O Vocabulário do Novo Testamento Grego*.                                      |
| **n**                   | Nota na Referência                             | Indica que existe uma nota pertinente nesta referência.                                                              |
| **or,** (itálico)       | Leitura Alternativa (Suprida pelo Tradutor)    | Leitura marginal fornecida pelos tradutores da versão KJV.                                                           |
| **or,** (não itálico)   | Renderização Alternativa (Suprida pelo Editor) | Renderização alternativa da Tradução Literal de Robert Young.                                                        |
| **S#**                  | Concordância de Strong                         | Referências selecionadas aos números da Concordância de Strong.                                                      |
| **T#**                  | Número Tópico                                  | Indica um arranjo tópico das Escrituras.                                                                             |
| **TDNT**                | Dicionário Teológico do Novo Testamento        |                                                                                                                      |
| **lit.**                | Literalmente                                   |                                                                                                                      |
| **mg**                  | Leitura Marginal                               | Referência às leituras marginais na versão KJV/Autorizada.                                                           |
| **MM**                  | Moulton & Milligan                             | James Hope Moulton e George Milligan, *O Vocabulário do Novo Testamento Grego*.                                      |
| **n**                   | Nota na Referência                             | Indica que existe uma nota pertinente nesta referência.                                                              |
| **or,** (itálico)       | Leitura Alternativa (Suprida pelo Tradutor)    | Leitura marginal fornecida pelos tradutores da versão KJV.                                                           |
| **or,** (não itálico)   | Renderização Alternativa (Suprida pelo Editor) | Renderização alternativa da Tradução Literal de Robert Young.                                                        |
| **S#**                  | Concordância de Strong                         | Referências selecionadas aos números da Concordância de Strong.                                                      |
| **T#**                  | Número Tópico                                  | Indica um arranjo tópico das Escrituras.                                                                             |
| **TDNT**                | Dicionário Teológico do Novo Testamento        |                                                                                                                      |
| **w**                   | Com                                            | Indica relações entre passagens não na ordem bíblica.                                                                |
| **‡**                   | Doutrina Falsa                                 | Indica textos de prova usados para apoiar doutrinas falsas.                                                          |
| **? ou x**              | Validade Duvidosa                              | Indica uma referência questionável, uma citação mal identificada ou uma interpretação incorreta.                     |

Esta tabela deve ser útil para navegar e entender a riqueza de informações contidas no The New Treasury of Scripture Knowledge. É projetada para ser usada como referência no Obsidian para consulta rápida destes símbolos e abreviaturas.
