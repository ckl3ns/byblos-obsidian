## Como usar

- Após cada sessão de estudo: registrar insights em `{dominio}/hypotheses.md`
- Quando uma hipótese for confirmada 3+ vezes por fontes independentes de Nível 2-3: mover para `rules.md`
- Quando uma rule for contradita por evidência Nível 1: rebaixar para `hypotheses.md`
